# react-todo-list
Todo list made in React

Live App: https://murilloves.github.io/react/todolist/

## #1 - Install packages
```sh
$ npm i
```

## #2 - Start the app
```sh
$ npm start
```

It should be listening at [localhost:8080]

## #3 - Building the app
```sh
$ npm run build
```

It'll generate a folder named `prod`

[localhost:8080]: http://localhost:8080
